# 📚 Atividades de Banco de Dados (MySQL)

Repositório com exercícios e práticas utilizando **MySQL**, referente à UC de **Fundamentos de Banco de Dados**, ministrada pelo professor Valentim.

## 🧠 Conteúdo

- Criação de bancos e tabelas  
- Comandos SQL (`SELECT`, `INSERT`, `UPDATE`, `DELETE`)  
- `JOINs`  
- Funções agregadas  
- Normalização básica

## 🎯 Objetivo

Praticar SQL e reforçar fundamentos de banco de dados usando MySQL.
